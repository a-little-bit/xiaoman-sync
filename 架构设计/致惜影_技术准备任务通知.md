## 🎯 关键技术点说明

### 1. PostgreSQL + Citus 分库分表
```sql
-- 启用Citus扩展
CREATE EXTENSION IF NOT EXISTS citus;

-- 创建分片表
CREATE TABLE users (
    user_id BIGSERIAL,
    username TEXT,
    created_at TIMESTAMPTZ DEFAULT NOW()
) PARTITION BY HASH (user_id);

-- 设置分片数 (初始8片，支持扩容)
SELECT create_distributed_table('users', 'user_id', shard_count => 8);
```

### 2. Redis Cluster 配置
```yaml
# redis-cluster.yml
cluster-enabled: yes
cluster-config-file: nodes.conf
cluster-node-timeout: 15000
appendonly: yes
```

### 3. LBS地理位置查询
```sql
-- 附近的人查询 (PostGIS)
SELECT user_id, ST_Distance(
    location, 
    ST_MakePoint(116.404, 39.915)::geography
) AS distance
FROM user_locations
WHERE ST_DWithin(
    location,
    ST_MakePoint(116.404, 39.915)::geography,
    1000  -- 1000米范围
)
ORDER BY distance
LIMIT 100;
```
