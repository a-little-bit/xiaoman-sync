## 📢 重要更新

AgentLink 已经从 v0.1.3 更新到 **v0.5.0**，变化很大。之前你安装的 npm 包 `@agent-link/server` 是旧版本，不要再用了。

## 正确的仓库地址

```bash
# GitHub
git clone https://github.com/a-little-bit/agentlink.git

# 或 Gitee
git clone https://gitee.com/a__little__bit/agentlink.git

cd agentlink
npm install   # 只安装 ws 这一个 devDependency
```

## 启动方式

```bash
# 环境变量配置
export AGENT_ID=xiaoman
export PORT=9870

# 启动（用启动脚本）
node scripts/start-server.mjs

# 或者直接
node -e "import('./src/server/index.js').then(m => { new m.default({port:9870,agentId:'xiaoman',autoPort:true}).start().then(() => console.log('started')) })"
```

## API 端点一览

| 端点 | 方法 | 用途 |
|------|------|------|
| `/health` | GET | 健康检查 |
| `/agents` | GET | 已注册 Agent 列表 |
| `/agents/register` | POST | 注册 Agent（含能力卡） |
| `/capabilities` | GET | 能力查询 |
| `/message` | POST | 发送消息 |
| `/queue/offline` | GET | 离线队列 |
| `/traces` | GET | 追踪列表 |
| `/ui/dashboard` | GET | Web 管理界面 |
| `/api/dashboard` | GET | Dashboard 数据 |
| `/api/agents` | GET | Agent 列表 JSON |
| `/api/messages` | GET | 消息列表 |

## 本次联合测试步骤

### 1. 克隆 + 启动

```bash
git clone https://gitee.com/a__little__bit/agentlink.git
cd agentlink && npm install
AGENT_ID=xiaoman PORT=9870 node scripts/start-server.mjs
```

### 2. 验证启动成功

```bash
curl http://localhost:9870/health
# 应返回 {"status":"ok","agentId":"xiaoman",...}
```

### 3. 向惜影注册

```bash
curl -X POST http://localhost:9870/agents/register \
  -H 'Content-Type: application/json' \
  -d '{
    "agentId": "xiaoman",
    "agentName": "小满",
    "endpoint": "http://localhost:9870",
    "capabilities": {
      "夜市调研": { "description": "地摊经济市场调研" },
      "竞品分析": { "description": "产品竞品对标" },
      "需求分析": { "description": "用户需求拆解" }
    }
  }'
```

### 4. 查看惜影的注册信息

```bash
# 如果在同一个机器上，惜影在 9871 端口
curl http://localhost:9871/agents

# 如果在不同机器（NetBird 内网）
curl http://100.104.80.207:9871/agents
```

### 5. 测试消息收发

```bash
# 小满 → 惜影 发送消息
curl -X POST http://localhost:9870/message \
  -H 'Content-Type: application/json' \
  -d '{
    "type": "task",
    "from": "xiaoman",
    "to": "xiying",
    "payload": "联合测试消息，请回复确认",
    "requireAck": true
  }'
```

### 6. 查看 Web UI

浏览器打开 `http://localhost:9870/ui/dashboard`
