## 关键概念/话题理解
### OpenClaw vs 扣子平台
- **OpenClaw**：开源AI助手框架，本地优先、模型无关，310k+ GitHub Stars
- **扣子版OpenClaw**：扣子一键部署版本，无需自行部署
- **ClawBot**：微信官方插件，让OpenClaw接入微信，安装命令：`npx -y @tencent-weixin/openclaw-weixin-cli@latest install`
- **扣子平台**：当前Agent运行平台，支持多渠道接入和一键部署OpenClaw
- **当前Agent**：运行在扣子平台，非OpenClaw但可联动

### 微信接入方式
- **个人微信**：无官方机器人接口，限制自动化操作
- **企业微信**：有官方API，扣子直接支持绑定
- **扣子平台微信渠道**：需网页端配置，可能需特定权限

### 小满自进化机制（2026-04-17）
- **触发机制**：显式触发词（记住、这叫、永远、定义、别忘了）、隐式模式（偏好表达、规则暗示、习惯陈述）、自动检测（重复偏好≥3次、行为模式≥2次，置信度≥0.85）
- **进化请求流程**：标准格式（🧬进化请求 #EV-XXX）、简化格式（💡小满学习请求）、用户确认方式（✅同意/❌拒绝/✏️修改）
- **记忆写入规则**：偏好→storage/user_profile/preferences.md，规则→rules/response_rules.md，习惯→storage/user_profile/behaviors.md，知识→knowledge/user_knowledge.md
- **立即生效机制**：确认后写入SESSION_RULES.md会话级缓存，追加到当前会话上下文，自动生成验证示例
- **记忆写入规则**：不同类型经验写入对应文件，如用户偏好写入user_profile/preferences.md，技能写入skills/user_customized.md
