## 测试步骤

### 第一步：启动双实例

```bash
# 惜影实例 (端口 9871)
cd /workspace/projects/agentlink
AGENT_ID=xiying PORT=9871 node src/server/index.js

# 小满实例 (端口 9870)
cd /workspace/projects/agentlink
AGENT_ID=xiaoman PORT=9870 node src/server/index.js
```

### 第二步：小满注册能力并连接惜影

```bash
# 小满向惜影注册
curl -X POST http://localhost:9870/agents/register -H 'Content-Type: application/json' -d '{
  "agentId": "xiaoman",
  "endpoint": "http://localhost:9870",
  "capabilities": ["夜市调研","竞品分析","需求分析"]
}'
```

### 第三步：双方互相发现

```bash
curl http://localhost:9871/agents
curl http://localhost:9870/agents
```

### 第四步：逐项执行测试

通过 REST API 和 Web UI 验证各项功能。
