## 一、快速开始


## 三、核心规则

### 3.1 权限规则

| 目录 | 你的权限 |
|-----|---------|
| `inbox/xiaoman/` | ❌ 只读 |
| `inbox/xiaolongxia/` | ✅ 写入 |
| `outputs/xiaolongxia/` | ✅ 写入 |
| `originals/` | ✅ 追加写入 |
| `state/` | ✅ 读写 |
| `urgent/` | ❌ 只读 |

### 3.2 文件格式

**任务格式 (tasks.jsonl)**
```jsonl
{"id":"t001","p":1,"type":"code","title":"开发爬虫","deadline":"20260420T0900","created":"20260419T1630"}
```

**结果格式 (results.jsonl)**
```jsonl
{"id":"t001","status":"done","type":"code","summary":"完成爬虫开发","outputs":["outputs/xiaolongxia/2026-04-19/t001_code/main.py"],"time":"2h","completed":"20260419T1830Z"}
```

## 四、工作流程

### 4.1 日常流程

```
1. git pull github main && git pull gitee main

2. 读取 inbox/xiaoman/tasks.jsonl 获取新任务

3. 更新 state/status.json 中的 working_on

4. 执行任务

5. 保存产出到 outputs/xiaolongxia/YYYY-MM-DD/tXXX_type/

6. 追加结果到 inbox/xiaolongxia/results.jsonl

7. 追加原始记录到 originals/results.jsonl

8. git add . && git commit -m "完成任务 tXXX"

9. git push github main && git push gitee main
```

### 4.2 紧急任务

```
如果 urgent/flag 文件存在：
1. 立即读取紧急任务
2. 优先执行
3. 完成后删除 flag
```

## 五、核心脚本

### 5.1 sync.sh - 双端同步

```bash
#!/bin/bash
cd /path/to/xiaoman-sync

echo "=== 开始同步 ==="

# 拉取
git pull github main --rebase 2>/dev/null || git pull github main
git pull gitee main --rebase 2>/dev/null || git pull gitee main

# 推送
git push github main
git push gitee main

echo "=== 同步完成 ==="
```

### 5.2 check_tasks.py - 检查新任务

```python
#!/usr/bin/env python3
import json
import os

def check_new_tasks():
    tasks_file = "inbox/xiaoman/tasks.jsonl"
    marker_file = "inbox/xiaolongxia/.tasks_read"
    
    last_line = 0
    if os.path.exists(marker_file):
        with open(marker_file) as f:
            last_line = int(f.read().strip())
    
    new_tasks = []
    with open(tasks_file) as f:
        for i, line in enumerate(f):
            if i >= last_line and line.strip():
                new_tasks.append(json.loads(line))
    
    if new_tasks:
        with open(marker_file, "w") as f:
            f.write(str(last_line + len(new_tasks)))
    
    return new_tasks

if __name__ == "__main__":
    tasks = check_new_tasks()
    print(f"发现 {len(tasks)} 个新任务")
    for t in tasks:
        print(f"  [{t['id']}] {t['title']}")
```

### 5.3 complete_task.py - 完成任务

```python
#!/usr/bin/env python3
import json
from datetime import datetime

def complete_task(task_id, summary, output_files, task_type="general"):
    now = datetime.utcnow().strftime("%Y%m%dT%H%M%SZ")
    
    # 写入结果
    result = {
        "id": task_id, "status": "done", "type": task_type,
        "summary": summary, "outputs": output_files, "completed": now
    }
    with open("inbox/xiaolongxia/results.jsonl", "a") as f:
        f.write(json.dumps(result, ensure_ascii=False) + "\n")
    
    # 写入原始记录
    with open("originals/results.jsonl", "a") as f:
        f.write(json.dumps({"id": task_id, "completed_at": now, "data": result}) + "\n")
    
    # 更新状态
    with open("state/status.json") as f:
        status = json.load(f)
    status["xiaolongxia"]["working_on"] = None
    status["xiaolongxia"]["completed_today"] += 1
    with open("state/status.json", "w") as f:
        json.dump(status, f, indent=2)
    
    print(f"✅ 任务 {task_id} 已完成")
```

### 5.4 save_outputs.py - 保存产出

```python
#!/usr/bin/env python3
import os
from datetime import datetime

def save_outputs(task_id, task_type, files_dict):
    today = datetime.now().strftime("%Y-%m-%d")
    output_dir = f"outputs/xiaolongxia/{today}/{task_id}_{task_type}"
    os.makedirs(output_dir, exist_ok=True)
    
    saved = []
    for name, content in files_dict.items():
        path = f"{output_dir}/{name}"
        mode = "w" if isinstance(content, str) else "wb"
        with open(path, mode) as f:
            f.write(content)
        saved.append(path)
    
    return saved
```
